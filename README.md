# GregTech_Machines_Addon
A small mod to make your GregTech a bit easier.

# How to use

BUILD WITH CMD
--------------

Open `cmd` and run `git clone https://github.com/Texas-albe/GregTech_Machines_Addon`

Redirect to Folder `GregTech_Machines_Addon` and run `gradlew build`

Put the built jar file into your minecraft mod Folder `mods`

BUILD WITH DOWNLOADED FILE
--------------------------

Download the source code

Redirect to Folder `GregTech_Machines_Addon` and run `gradlew build`

Put the built jar file into your minecraft mod Folder `mods`
